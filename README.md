# GIS RAG Search Engine (Streamlit App)

This is a lightweight Retrieval-Augmented Generation (RAG) search engine
for GIS learning materials. It uses:

- SentenceTransformers (all-MiniLM-L6-v2)
- FAISS vector search
- Streamlit UI
- Scraped static GIS pages
